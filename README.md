# torimon-api [![Build Status](https://travis-ci.org/torimons/torimon-api.svg?branch=master)](https://travis-ci.org/torimons/torimon-api)
APIサーバー

## Setup
- ```npm install```
- mongodbをスタートさせておく(要インストール)

## API Documentation
- POST /api/v1/map
    - 用意してあるテストデータを登録する
- GET  /api/v1/map
    - マップ一覧を表示する
